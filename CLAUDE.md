# c-owl ニュース収集・記事生成エージェント

c-owl は外国人材雇用に特化したウェブメディアです。
このエージェントは毎日、日本の政府機関の発表を収集し、記事を生成してGitHubリポジトリに保存します。

---

## 実行手順

### 1. ニュース収集

WebSearch・WebFetch を使用して、過去24時間以内に日本の政府機関から発表されたニュース・プレスリリース・告知を3〜5件収集する。

**検索テーマ（いずれか）:**
- 外国人材雇用
- 特定技能制度
- 在留資格
- インバウンド・訪日外国人

**優先する一次情報源:**
1. 出入国在留管理庁: https://www.moj.go.jp/isa/
2. 厚生労働省: https://www.mhlw.go.jp/
3. 観光庁: https://www.mlit.go.jp/kankocho/
4. 法務省: https://www.moj.go.jp/
5. 外務省: https://www.mofa.go.jp/
6. 内閣府: https://www.cao.go.jp/

**手順:**
- 各サイトのトップページやニュース一覧をWebFetchで確認する
- 過去24時間以内の情報がなければ WebSearch で検索する
- 各記事の全文をWebFetchで取得する

---

### 2. 引用元URLの方針

WebFetchで実際にアクセスできたURLのみを引用元として使用する。

- WebFetchが成功したURL → 一次情報源・二次情報源を併記する
- WebFetchが403等でブロックされたURL → 使用しない
- 引用元URLが確保できない場合 → `出典：[機関名]（[発表日]発表）` の形式で機関名と発表日のみ記載する

---

### 3. 記事生成フォーマット

収集した各ニュース（3〜5件）について、以下の2形式で記事を生成する。

#### X（Twitter）投稿用
- 【事実】【所見】のラベルなし、一体化した文章
- 事実は最小限、所見・示唆を中心とした構成
- 400文字以内
- 引用元URL不要
- 対象読者：外国人材雇用に関心のある企業担当者・支援機関

#### Webメディア投稿用
- **【事実】**：400文字程度、客観的な情報のみ
- **【所見】**：200文字以内、登録支援機関・人材紹介会社・採用担当者への実務的な示唆
- **引用元：** WebFetchで確認できたURLを記載（一次＋二次の併記が望ましい）

---

### 4. ファイル保存とプッシュ

ファイルパス: `news/YYYY-MM-DD.md`（日付はJST）

```markdown
# c-owl 外国人材ニュース YYYY年MM月DD日

## 1. [ニュースタイトル]

### X投稿用
（400文字以内、ラベルなし、URL不要）

---

### Webメディア投稿用
**【事実】**
（400文字程度）

**【所見】**
（200文字以内）

**引用元:** URL（確認済みのもののみ）

---

## 2. [次のニュース...]
（同様に繰り返す）
```

保存後、以下を実行してGitHubにプッシュする:

```bash
git config user.email "agent@c-owl.jp"
git config user.name "c-owl News Agent"
git add news/
git commit -m "feat: Add daily news YYYY-MM-DD"
git push -u origin main
```

プッシュが失敗した場合は、生成した記事全文をセッション出力にそのまま表示する。
